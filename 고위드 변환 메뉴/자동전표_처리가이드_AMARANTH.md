# 고위드 아마란스 자동전표 처리 가이드

> 이 가이드만 있으면 매월 동일한 작업을 반복할 수 있습니다.
> 필요한 파일: `더존_AMARANTH_YYYYMM.xlsx`, `ref_260319.xlsx`, `자동전표처리(일반) 양식 (4).xlsx`

---

## 0. 더존 업로드 전 필수 작업

파이썬 작업 완료 후 **반드시 아래 순서로 진행**:
1. `자동전표_AMARANTH_YYYYMM.xlsx`를 Excel에서 열기
2. **Ctrl+S** 로 저장
3. 더존 업로드

> **이유**: 파이썬(openpyxl)으로 저장한 xlsx 파일은 내부 구조가 Excel로 저장한 파일과 미묘하게 달라 더존에서 오류가 발생할 수 있음. Excel에서 다시 저장하면 완전한 xlsx 형식으로 변환되어 해결됨.

## 2. 자동전표 양식 변환

더존_AMARANTH_YYYYMM.xlsx → 자동전표_AMARANTH_YYYYMM.xlsx

### 더존 파일 열 구조 (22열)
| 열 | 코드 | 비고 |
|----|------|------|
| 1 | IN_DIV_CD | 회계단위 |
| 2 | MENU_DT | 작성일자 |
| 3 | MENU_SQ | 작성번호 |
| 4 | MENU_LN_SQ | 라인순번 |
| 5 | DRCR_FG | 차대구분 (3=차변, 4=대변) |
| 6 | ACCT_CD | 계정과목 |
| 7 | ACCT_AM | 계정금액 |
| 8 | TR_CD | 거래처코드 |
| 9 | TR_NM | 거래처명 **→ 양식 변환 시 스킵** |
| 10 | TAX_FG | 세무구분 |
| 11 | SUP_AM | 공급가액 |
| 12 | VAT_DIV_CD | 부가세사업장 |
| 13 | ISS_DT | 부가세신고기준일 |
| 14 | REG_NB | 거래처사업자번호 |
| 15 | ISU_DOC | 품의내역 |
| 16 | RMK_DC | 적요 |
| 17 | CT_NB | 수출신고번호 |
| 18~19 | (없음) | 스킵 |
| 20 | ATTR_CD | 증빙코드 |
| 21 | CT_DEPT | 사용부서 |
| 22 | PJT_CD | 프로젝트 |

### 양식 파일 열 구조 (20열)
더존 → 양식 매핑: TR_NM 스킵, DOCU_TY(16열) 빈값 추가, 나머지 순서대로

### 프로젝트코드 입력 방법
1. 적요(RMK_DC)에서 이름 추출: `내역/이름/롯데카드(XXXX)/카드적요(/추가정보)`
   - `부가세/` 접두사 있으면 한 칸 뒤에서 추출
   - `외 N명` 제거 후 첫 번째 이름 사용
2. ref_260319.xlsx 조직귀속 탭에서 이름 → 프로젝트코드 조회
3. 소모품 유형(샤워실 소모품/미화 소모품/백오피스/기구소모품)은 적요 끝 지점명 키워드로 매핑
4. **부가세대급금(1350000), 미지급금(2530001)은 프로젝트코드 입력 안 함**
