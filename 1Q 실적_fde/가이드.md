### cost_overrides.json 형식
특정 거래를 다른 지점으로 재분류할 때 사용
```json
[
  {
    "from_branch": "광화문",
    "to_branch": "판교",
    "is_tb": true,
    "귀속연월": "2026-01",
    "의뢰인": "우미회관"
  }
]
```
→ 광화문의 2026-01 우미회관 팀활동비를 판교 TB로 이관

### API 엔드포인트
| 경로 | 설명 |
|------|------|
| `GET /api/cost/{branch}/pivot` | 지점 피벗 데이터 |
| `GET /api/cost/{branch}/detail?소분류=xxx&월=2026-01` | 상세 내역 |
| `GET /api/cost/download?year=2026` | 전지점 엑셀 다운로드 |

### 이상치 감지 기준
- **값 누락**: 다른 달 2개 이상에 값 있고 평균 5만원 이상인데 이 달이 0
- **급등**: 다른 달 평균의 3배 이상이고 절대값 10만원 이상
- 확인완료: localStorage에 저장 (`cost_ok__{branch}__{소분류}__{월}__{금액}`)

## 5. FDE 배포 (미완료 — 다음 작업)

### 완료된 파일
- `fde/backend/fde/static/cost.html` ← FDE API 경로로 수정된 HTML
  - `BASE = '/fde-api/jihee/cost'`로 변경됨
  - fetch 경로: `/branches`, `/{branch}/pivot`, `/{branch}/detail`, `/download`

### 남은 작업

**1. 백엔드 라우터 생성**
`fde/backend/fde/routers/jihee_cost.py` 신규 생성

FastAPI 라우터 (Flask server.py의 비용 대시보드 로직을 FastAPI로 이식)

엔드포인트:
- `GET /branches`
- `GET /{branch}/pivot`
- `GET /{branch}/detail`
- `GET /download`
- `GET /dashboard` → static/cost.html을 HTMLResponse로 서빙

서비스 계정 파일 경로:
```python
BASE_DIR = os.path.dirname(os.path.abspath(__file__))  # routers/
FDE_DIR  = os.path.dirname(BASE_DIR)                   # fde/
SERVICE_ACCOUNT_FILE = os.path.join(os.path.dirname(FDE_DIR), "sales_data", "service_account.json")
COST_OVERRIDES_FILE  = os.path.join(FDE_DIR, "cost_overrides.json")
STATIC_DIR           = os.path.join(FDE_DIR, "static")
```

**2. main.py 수정**
```python
# import 추가
from routers import ..., jihee_cost

# 인증 면제 추가
_AUTH_EXEMPT_PREFIXES = {
    "/fde-api/sales",
    "/fde-api/jihee/revenue",

# 라우터 등록 추가

```

**3. cost_overrides.json 복사**
`매출보고 대시보드/cost_overrides.json` → `fde/backend/fde/cost_overrides.json`

**4. React 프론트엔드 카드 추가**
`fde/frontend/packages/erp/src/pages/ChoiJihee/index.tsx`에 카드 추가:
```tsx

```

```bash

./deploy.sh fde-backend
./deploy.sh erp
```
